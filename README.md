Jeu de Quiz
Un petit jeu de quiz interactif en Python qui pose des questions aléatoires et évalue vos réponses. Testez vos connaissances et amusez-vous !

Description
Le programme pose une série de questions choisies aléatoirement. L'utilisateur répond à chaque question et reçoit un score basé sur ses réponses correctes. À la fin, le score total est affiché avec un message d'encouragement.

Fonctionnalités principales
Mélange aléatoire des questions.
Comparaison des réponses sans distinction de majuscules/minuscules.
Calcul du score final.
Feedback personnalisé basé sur la performance.
Installation
Assurez-vous d'avoir Python installé sur votre machine (Python 3.6 ou une version ultérieure est recommandé).
Téléchargez ou clonez ce dépôt.
Exécution du projet
Pour exécuter le jeu de quiz, ouvrez un terminal ou une console, puis lancez la commande suivante dans le répertoire du projet :
