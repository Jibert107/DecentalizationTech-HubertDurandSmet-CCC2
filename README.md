# Quiz Game

Un jeu de quiz simple en Python qui pose des questions aux utilisateurs et évalue leurs réponses.

## Description

Ce programme est un jeu de quiz interactif qui :
- Pose une série de questions à l'utilisateur
- Vérifie les réponses
- Calcule le score final
- Donne un retour personnalisé selon la performance
